# 行为准则

本项目采用 Contributor Covenant 行为准则。

## 我们的承诺

为了营造开放且友好的环境，我们承诺：无论年龄、体型、残障、族裔、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向如何，我们都会尊重每一位参与者。

## 我们的标准

有助于创造积极环境的行为包括：

- 友善且包容的语言
- 尊重不同观点与经验
- 接受建设性反馈
- 关注对社区最有利的结果
- 对其他社区成员表现同理心

不可接受的行为包括：

- 人身攻击、侮辱或贬损性评论
- 公共或私下的骚扰
- 未经同意发布他人隐私信息
- 其他在专业环境中不适当的行为

## 责任范围

本行为准则适用于项目空间内的一切互动，包括但不限于：Issue、Pull Request、讨论区、代码审查以及官方社区渠道。

## 处理与执行

若你遇到或目睹违规行为，请通过 `SECURITY.md` 中提供的联系方式进行报告。维护者会根据情况采取适当且公平的处理措施。
